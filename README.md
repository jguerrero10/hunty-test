
# Hunty App - Test

Prueba Técnica

## Instalación
- Docker debe estar instalado, para verificar ejecutar: 

``` $ docker run hello-world ```

- Para construir la imagen ejecutar:

``` $ docker-compose build ```

- Para levantar la imagen:

``` $ docker-compose up -d ```
